# 发票自动筛选报销系统

## 功能概述
从电子发票(PDF/图片)中自动识别发票信息，并根据深圳前海微众银行月度福利补贴规则，筛选出当月可报销的发票。

## 系统架构

```
invoice_reimbursement/
├── README.md                 # 本文档
├── requirements.txt          # 依赖包
├── config.py                # 配置文件
├── main.py                  # 主入口
├── models.py                # 数据模型
├── ocr/
│   ├── __init__.py
│   ├── base.py              # OCR基类
│   ├── paddle_ocr.py        # PaddleOCR实现
│   └── tesseract_ocr.py     # Tesseract实现
├── validators/
│   ├── __init__.py
│   ├── base.py              # 校验基类
│   ├── invoice_validator.py # 发票规则校验
│   └── company_rules.py     # 公司规则配置
└── output/
    └── invoices/            # 待识别发票目录
    └── reports/             # 报销结果输出
```

## 依赖安装

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
# 将发票放入 output/invoices/ 目录
python main.py --month 2026-01 --employee "张三"
```

## 报销规则（基于微众银行规定）

### 1. 时间规则
- 发票开票日期或费用发生周期应在**3个月内**
- 例：报销1月份补贴，发票日期需在11月-1月之间

### 2. 发票类型规则

#### 餐饮发票 ✅
- 公司抬头的电子/纸质发票
- 超市购买食品需附销货清单
- ❌ 非食品类不可报销（如充值权益）

#### 交通发票 ✅
- 本人名字的高铁票、机票
- 公司抬头的加油票、停车费、电车充电、过路费、通行费
- 出租车票/网约车发票
- 定额发票（停车票、地铁充值票）

#### 通讯发票 ✅
- 员工个人名字的发票（含手机话费、网费）
- 公司抬头的发票
- 定额发票

### 3. 发票抬头信息

**普票（必填）：**
- 名称：深圳前海微众银行股份有限公司
- 纳税人识别号：9144030031977063XH

**专票（必填全部）：**
- 名称、纳税人识别号、开户行、账号、地址、电话

### 4. 异常开票方黑名单
以下开票方不可报销：
- 青浦金泽状元楼饭店
- 武汉肖记关山坊牛肉杂鱼馆
- 山城记·爆炒川菜馆
- 烫锅鲜砂锅串串
- 等（详见 company_rules.py）
