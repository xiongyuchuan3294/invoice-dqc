# 发票智能筛选系统

从电子发票(PDF)中自动识别发票信息，并根据月度福利补贴规则，智能筛选出当月可报销的发票。

## 功能特点

- **自动识别**：从PDF发票中提取关键信息（类型、金额、日期、销方等）
- **智能校验**：自动校验发票抬头、日期范围、黑名单等
- **FIFO选择**：按日期优先选择满足补贴上限的发票
- **自动归档**：选中发票移入月份目录，错误发票移除，多余发票保留

## 补贴金额上限

| 类型   | 上限   |
| ------ | ------ |
| 餐饮   | 680元  |
| 交通   | 500元  |
| 通讯   | 300元  |

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
# 将发票放入 invoice/ 目录
python main.py --month 2026-01
```

### 参数说明

- `--month`：报销月份，格式 YYYY-MM（必填）
- `--invoice-pool`：发票池目录（默认: invoice）
- `--output-dir`：输出目录（默认: output）

## 输出结构

```
invoice/                         # 原始发票池
├── [多余]餐饮_500.00元_某某公司.pdf     # 多余发票（保留，后续可用）
└── ...

output/
├── 2026-01/                              # 选中发票（按月份归档）
│   ├── 餐饮_110.00元_武汉市渔焰餐饮有限公司.pdf
│   ├── 交通_50.00元_滴滴出行科技有限公司.pdf
│   └── 通讯_100.00元_中国移动通信集团.pdf
├── errors/                               # 错误发票
│   ├── [错误-超期]餐饮_399.00元_某某公司.pdf
│   └── ...
└── 报销报告_202601_20250125_120000.xlsx   # 报销报告
```

## 文件命名规则

| 状态 | 命名格式 |
|------|----------|
| 选中 | `{类型}_{金额}元_{销方名称}.pdf` |
| 多余 | `[多余]{类型}_{金额}元_{销方名称}.pdf` |
| 错误 | `[错误-{原因}]{类型}_{金额}元_{销方名称}.pdf` |

## 报销规则

### 1. 时间规则
- 发票开票日期应在**3个月内**
- 例：报销1月份补贴，发票日期需在11月-1月之间

### 2. 发票抬头
- 普票：公司名称 + 纳税人识别号
- 专票：名称 + 税号 + 开户行 + 账号 + 地址 + 电话

### 3. 可报销类型
- **餐饮**：公司抬头的电子发票；超市购物需附清单；非食品类不可报销
- **交通**：本人名字高铁/机票；公司抬头加油/停车/充电；出租车/网约车发票
- **通讯**：员工个人名字或公司抬头的发票

### 4. 黑名单
- 青浦金泽状元楼饭店
- 武汉肖记关山坊牛肉杂鱼馆
- 山城记·爆炒川菜馆
- 等（详见 config.py）
